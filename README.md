# 💼 Job Portal

A modern Full Stack MERN Job Portal application that connects job seekers with employers. Users can search and apply for jobs, save favorite jobs, and manage their profiles, while employers can post, edit, and manage job listings through a dedicated dashboard.

## 🚀 Features

- 🔐 JWT Authentication
- 👥 Role-based Access (Job Seeker & Employer)
- 🔍 Search & Filter Jobs
- 💾 Save Jobs
- 📄 Apply for Jobs
- 🏢 Employer Dashboard
- ➕ Create Job Listings
- ✏️ Edit & Delete Jobs
- 👤 User Profile Management
- 📱 Responsive Design

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/iamroysunny/Job-Portal.git
```

### Install dependencies

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm run dev
```
